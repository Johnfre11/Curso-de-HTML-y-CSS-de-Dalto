RUTAS en HTML

Descripcion del apartado

En este apartamos vamos a explicar como funcionan las rutas para llamar distintos archivos que se encuentren en carpetas diferentes.

Tipos de rutas

  • Rutas de tipo ABSOLUTAS:

    • No cambian nunca por más de que este archivo cambie de lugar o de ruta, por ejemplo: la ruta del video con el que estoy aprendiendo HTML.

    Curso de HTML y CSS = https://youtu.be/ELSm-G201Ls?t=5588

  • Rutas de tipo RELATIVA:

    • Rutas que dependen de la ubicación del archivo que se está llamando, por ejemplo, en donde tengo guardado este archivo .html tengo una imagen llamada Riven_vs_Yasup pero en el momento que la guarde en la carpeta de imagnes esta va a dejar de funcionar porque ya no se encuentra en la misma ubicación de este archivo .html

Manejos de rutas relativas.

Una buena practica es usar rutas relativas en el proyecto de la página web, pero entoncos ¿comó puedo accerder a la imagen anterior que no estaba cargando?

Por ejemplo:

Mi carpeta de este proyecto es tiene esta estructura

📁 Carpeta_del_proyecto

->📂CH_03_enlaces_anclas_0 / 📄CH_03_enlaces_anclas_1.html

->📂CH_04_imagenes_0 / 📷Riven_vs_Yasup.png

->📂CH_05_html_0 / ->📄Este_archivo.html<-

Para llegar hasta la imagen primero necesito salir de la carpeta "📂CH_05_html_0" para esto puedo usar "../" lo que significa que se va a retroceder una carpeta, en caso de que esté más carpetas atrás se puede repetir la sintaxis, por ejemplo, dos carpetas atrás sería "../../"

Ahora que se está en la carpeta principal "Carpeta_del_proyecto" se puede aceder a la carpeta "📂CH_04_imagenes_0" y seleccionar la imagen en cuestión.

Se debe tener en cuenta que se arranca desde el archivo donde se está trabajando al archivo buscado, es decir, vamos a llevar el "📄Este_archivo.html" a "📷Riven_vs_Yasup.png"

Finalmente, quedaría algo así = ../📂CH_04_imagenes_0/📷Riven_vs_Yasup.png
