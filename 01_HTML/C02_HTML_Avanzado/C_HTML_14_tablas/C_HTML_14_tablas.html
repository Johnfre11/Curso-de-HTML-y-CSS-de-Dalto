Tablas

La etiqueta de las tablas es "tables", las tablas tienen sus etiquetas internas, las cuales son: "th" (table header) es el encabezado de las columnas, "tr" (table row) lo que hace es crear una fila, "td" crea celdas dentro de las filas

Otras etiquetas importantes de las tablas son: "thead" define el encabezado de la tabla, "tbody" define todos los datos de la tabla, "tfoot" define el final de la tabla, usualmente para totales.
