Etiquetas Obsoletas

Son etiquetas que se deben evitar usar, esto debido a que son obsoletas o la funcion se puede hacer con CSS

  • "font" (dar estilo a al texto)
  • "center" (centrar objetos)
  • "blink" (titilar objetos)
  • "big" (aumentar el tamaño de objetos)
  • "strike" (tachar texto), fue reemplazado con "del" y "s"
  • "tt" (convertir a tipografía específica de código), reemplzada por "code"
  • "u" (subraya texto)

Para estár en constante actualización de cuales son las etiquetas obsoletas se pueden averiguar poniendo en Google "mdn html" el cual es como el estandar oficial de html.
