Accesibilidad web

La accesibilidad web es para que las páginas puedan ser usadas por todo tipo de usuarios, incluyendo a las personas que no puedan ver. Además, hay países donde este tipo de accesibilidad es exigida por ley.

Para crear esta accesibilidad web se debe usar unos atributos como lo puede ser:

  • "alt" para imágenes.
  • "aria-label" para el resto de elementos.

Ejemplo de accesibilidad

  • Google

Este hipervínculo de Google tiene un "aria-label" que indica su funcionalidad para que la herramienta de accesibilidad se lo indique al usuario.

Roles

Los roles son importantes en la estructura de una página web, esto para que la herramienta de accesibilidad sepa identificar las secciones o roles de cada elemento.
