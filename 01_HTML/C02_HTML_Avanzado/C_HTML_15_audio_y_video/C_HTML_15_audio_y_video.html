Audio y video

Existen dos etiquetas para estas funciones las cuales son "video" y "audio".

Video

La etiqueta "video" tiene algunos atributos como: "autoplay" para que se reproduzca automaticamente cuando el usuario interactua con la página, "controls" para que se muestren los controles del video y "poster" para poner una miniatura del video, "muted" que funciona para que que haya una interacción y el video se reproduzca siempre que se cargue la página y el usuario le ponga volumen y "loop" lo que hace es reproducir el video en bucle.

Dentro de la etiqueta "video" se puede agregar otra llamada "track" para agregar subtitulos, esta etiqueta tiene distintos atributos como: "src" para agregar la ruta del archivo .vtt de los subtitulos, "default" para agregar los subtitulos por default, "kind" para definir el tipo, "srclang" para definirle al navegador en qué idioma están los subtitulos, "label" para agregar una etiqueta a los subtitulos. Para este ejemplo no tengo un arcivo .vtt pero dejará todo escrito como debería ser.

Audio

La etiqueta "audio" tiene los mismos atributos que la etiqueta "audio" menos el atributo "poster" no lo tiene porque los audios usualmente no tiene miniatura.
