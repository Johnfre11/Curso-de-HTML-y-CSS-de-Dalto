Display con Block

Para ver esta propiedad vamos utilizar dos enlaces (etiqueta "a"), por defecto estas etiquetas son elementos en línea y para cambiar esto usaremos "display", además, le agregaré un fondo para que se pueda apreciar mejor su comportamiento.

Como se puede observar los enlaces se ubicaron uno debajo de otro.

La propiedad "display" acompañada del valor "block" lo que hace es decirle a esa etiqueta que ocupe todo el ancho disponible como se puede observar con el color de fondo, además, todo lo que se encuentra después de la etiqueta arranca en una linea nueva.

Display con inline

Así como podemos convertir elemento en linea a elementos en bloque, podemos hacer lo contrario con la propiedad "display" acompañada del valor "inline". Por ejemplo, "p" es una etique que por defecto viene en bloque.
