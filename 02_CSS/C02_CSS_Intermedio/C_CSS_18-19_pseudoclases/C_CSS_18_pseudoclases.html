Input: Checked, valid, invalid

Para las etiquetas input hay varias psudoclases que se pueden usar.

Checked

Cambia una propiedad en caso de que la casilla esté seleccionada. En este caso, la casilla se mueve 20px a la derecha.

Valid e invalid

Comprueba que el formato del input sea correcto. Por ejemplo para un email, se pone de color rojo cuando en invalido y verde cuando es válido.

Links: link, visited

Comprueba que el link está visitado o sin visitar. Por ejemplo, se pone de color rojo cuando no está visitado y verde cuando está visitado.
