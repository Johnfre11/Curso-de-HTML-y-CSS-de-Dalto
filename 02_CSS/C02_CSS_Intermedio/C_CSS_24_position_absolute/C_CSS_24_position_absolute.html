Posición Absoluta

Cuando se le da la propiedad de "position" con el valor de "absolute" a un elemento, este elemento pasa a no ocupar un espacio en la página web, es decir, el elemento se puede observar pero no tiene un lugar específico en la página web, esta posición del elemento se la debemos dar con las propiedades de "top", "left", "bottom" y "right", respecto a la esquina superior izquierda.


Por ejemplo, creamos 3 cajas de dimensiones 100x100px, pero la caja 2 estará ubicada con posición absoluta.

Sin embargo, este punto de referencia se puede cambiar, haciendo que su contenedor tenga la propiedad de "position" con el valor "relative".
