Selectores Avanzados

Modificar atributos seleccionando uno en espeífico (SELECTORES POR ATRIBUTO).

Por ejemplo, la siguiente imagen tiene como source "src" igual a "Imagenes/imagen-de-prueba.png".

Para Modificar las propiedades de esta imagen se debe usar "[]" en css, por ejemplo, esta imagen le quiero cambiar únicamente el tamaña, por lo tanto en css debo escribir la siguiente sintaxis: img[src="Imagenes/imagen-de-prueba.png"].

También para seleccionar las imágenes que terminan en png se usaría de la siguiente manera: img[src$="png"] agregando el signo $ despues del atributo al que se quiere referir. El signo $ indica que se debe terminar con eso que se encuentre entre comillas.

El signo "^" funciona como signo para indicar algo con lo que empieza, por ejemplo, la misma imagen se puede modicar usando la siguiente sintaxis: img[src^="Imagenes"]

Modificar por hijos (SELECTORES POR HIJOS).

Para un ejemplo crearé una lista desordenada.

En la siguientelista tenemos la etiqueta "ul" que es el papá de todos los elementos que tiene dentro, los "li" que son hijos directos de "ul" y por último, la etiqueta "strong" que es hijo directo del tercer "li" y nieto de "ul".

  • Elemento 1: facil.
  • Elemento 2: normal.
  • Elemento 3: dificil

Para cambiar las propiedades de una etiqueta que sea hija directa o indirecta basta con escribir el nombre de la etiqueta padre y la hija (directa o indirecta). Por ejemplo, le voy a cambiar el color de la etiqueta "strong" con la siguiente sintaxis: ul strong{ }

Para cambiar las propiedades de una etiqueta que sea hija directa se debe escribir el nombre del padre seguido por el signo de mayor que ">" y por último el nombre de la hija (directa). Por ejemplo, voy a subrayar la etiqueta "li" con la siguiente sintaxis: ul li{ }, dado que "strong" es hija de "li" por lo tanto también adopta esta propiedad.

Por ejemplo, si yo quisiera tachar "strong" con la forma de hijo directo no podría porque "strong" no es hijo directo de "ul".

Modificar elementos adyacentes (SELECTOR ADYACENTE).

Para cambiar las propiedades de una etiqueta adyacente se debe escribir el primer elemento seguido por "+" y por último, el elemento adyacente. Por ejemplo, le voy a cambiar el color al "h2" que esté seguido por un "h1" con la siguiente sintaxis: h1 + h2{ }

Modificar elementos adyacentes (SELECTOR DE HERMANOS GENERALES).

Para cambiar las propiedades todas las etiquetas que vengan después de una etiqueta específica se realiza con el simbolo "~".Por ejemplo, le voy a cambiar el color a todos los "p" que vengan después de un "h2" con la siguiente sintaxis: h2 ~ p{ }
