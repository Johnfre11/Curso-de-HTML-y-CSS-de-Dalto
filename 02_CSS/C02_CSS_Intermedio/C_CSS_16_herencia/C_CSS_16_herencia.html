Herencia

Herencia es la cualidad de que las propiedades de un elemento se mantengan en los hijos. Por ejemplo:

Este parrafo se encuetra dentro de un "div" y además a este "div" le estoy cambiando en color de la letra, por lo tanto, este parrafo aunque no le estoy cambiando las propiedades debe ser color azul por la cualidad de herencia.

Este párrafo va a mostrar los valores de "inherit" e "initial", "inherit" es el valor de heredad del paradre e "initial" es dejar la propiedad por defecto, es decir, deja el valor que tiene el navegador. Por ejemplo, este parrafo mantiene el color porque tiene el valor de "inherit" pero no mantiene la familia porque tiene el valor de "initial".
