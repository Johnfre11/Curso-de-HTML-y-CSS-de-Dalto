Unit_em (hace un momento)

La medidad de em es una medida respecto al font-size del elemento padre. Por ejemplo, a este parrafo le voy a asignar un tamaño de 0.5em y como el elemento padre es el div al cual le asigné un tamaño de 20px, este párrafo debe tener un tamaño de 10px (20*0.5).
Pero, en caso de que el elemento padre no tenga definido un font-size, el tamaño rem iría a buscar el elemento padre siguiente, es decir, el padre del padre del elemento, y esto se irá repitiendo hasta que encuentre un font-sise de referencia hasta encontrar uno, en caso de que no encuentre ninguno irá hasta el elemento raiz, HTML.

Unit_rem (hace un momento)

La medidad de rem es una medida respecto al font-size del elemento raiz, a diferencia del em que busca al elemento padre. El elemto raiz siempre va a ser HTML. Por ejemplo, a este parrafo le voy a asignar un tamaño de 0.75rem y como el elemento raiz es HTML al cual le asigné un tamaño de font-size de 20px, este párrafo debe tener un tamaño de 15px (20*0.75)

Unit_vh_vw (hace un momento)

La medidad de vh y vw hace referencia al view port, es decir, al tamaño de la ventana donde se está viendo, es una medida se representa en porcentaje. Por ejemplo, el ancho (width) de este div tiene el 75% y el alto un 15% de la ventana del navegador.


Hay otras unidades del view port y son el vmin y vmax. La unidad vmin es el minínmo entre el ancho y el alto, por otro lado, vmax es el máximo entro el ancho y alto.
