Ejemplos de la propiedad Background-image

Ahora vamos a probar la función Background-image la cual dentro de un la etiqueta puede incluir una imagen de internet.
Por ejemplo:

Esto está bien, pero ahora tenemos un problema, dado que el div anterior donde tenemos el fondo con esa imagen de prueba se ve la imagen cortada porque no cabe en el tamaño del div que definí, por lo tanto viene la siguiente propiedad:

Background-size

Lo que hace esta propiedad es ajustar el ancho de la imagen respecto al ancho de la etiquita donde queremos ponere la imagen. Esta propiedad se puede medir como porcentaje o con otros valores de css que veremos más abajo, este porcentaje está refenciando al ancho de la etiqueta.
Por ejemplo, la primera imagen tiene 75% del ancho de la etiqueta, por lo cual el 25% restante sale cortado, y la segunda imagen tiene el 100% del ancho de la etiqueta, sin embargo, en la parte inferior se puede apreciar que se corta de una manera símilar a la imagen anterior en su ancho.

Valores especiales de Background-size

El primer valor es "contain" este valor lo que hace es ajustar la imagen de fondo a la etiqueta para que entre al menos una vez, este ajuste puede ser tanto para el alto como para el ancho.

El segundo valor especial es "cover" este valor lo que hace es ajustar la imagen sin que se repita lo mejor que se pueda al tamaño de la etiqueta, este valor se puede combinar bien con la propiedad background-position y sus valores: left, right, center, bottom, top; la propiedad de background-position también se puede usar con coordenadas.

El siguiente valor es background-attachment junto con el valor "fixed" lo que hace es mantener la imagen con su resolución original y fija, es decir que a medida que se navega por la página la etiqueta se mueve con la página pero la imagen no.
